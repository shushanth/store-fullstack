# pet-catalog-frontend

## Description

App displays the pet list with page actions and pet details.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run start (http://localhost:4000)
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Run Unit Tests

```sh
npm run test
```
